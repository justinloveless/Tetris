
ENTER - When game over,
	start new game
	
SPACE - Rotate piece

Arrow Left - Move piece left

Arrow Right - Move piece right

Arrow Down - Move piece down

Arrow Up - Drop piece 

M - Toggle Music ON/OFF

P - Pause/Start game

Q - Quit Game

PAUSED is now displayed when paused.

for a new block, but if you try to access
the save block, it will not let you. For
every new block that is called, you can
only switch with saved once.

is reduced.
